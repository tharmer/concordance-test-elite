# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.5.x | ✅ |
| 1.4.x | ✅ |
| < 1.4 | ❌ |

## Reporting a Vulnerability

Please report security vulnerabilities via email to security@example.com.

Do NOT open a public GitHub issue for security vulnerabilities.

We aim to acknowledge reports within 24 hours and provide a fix within 72 hours for critical issues.
